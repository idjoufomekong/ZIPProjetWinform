1. SI JobOverview
*****************

* Documents livrés:

- Logiciel
	- Archive zip du projet VS, à décompresser pour accéder à la solution (.zip)
	- Executable de l'application (.exe)
	- Fichier contenant les requètes utilisées pour la création des 
		tables utilisée pour l'insertion de masse (.sql)
	- Fichier de configuration (.config)

- Notice:
	- ReadMe (.txt)
	- Tutoriel (.pdf) 
	- Doc Technique (.pdf)
	- Schéma XML (.xsd)


- Sauvegarde de la base:
	- JobOverview (.bak)

* Initialisation de la BDD:

- Charger la sauvegarde de la base 'JobOverview.bak'

2. Lancement de l'application
*******************************

Partie technique :

- Charger la sauvegarde de la base de donnée
- Utiliser le fichier TypesTable.sql pour créer les tables dans votre base de donnée
	(dans la base ou vous avez charger la sauvegarde JobOverview (.bak))
- Renseigner le chemin de connexion dans les settings de l'application
	(voir tutoriel "Lancement")
	
Partie utilisateur :

- Utiliser le .exe pour lancer l'application
- Référez vous au tutoriel pour les détails d'utilisation
